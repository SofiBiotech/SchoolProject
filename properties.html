CSS Properties

Here are three CSS properties that can be used to format text:

  • Font Family - Specifies the font type used for text, such as Arial, Times New Roman, etc.
  • Font Size - Controls the size of the text, using units like pixels (px), em, rem, etc.
  • Text Align - Aligns text to the left, right, and center.
